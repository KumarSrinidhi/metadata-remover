- [ ] Read [README.md](README.md)

- [ ] Setup dev environment via [DEVELOPMENT.md](DEVELOPMENT.md)

- [ ] Install prerequisites (Java 17+, Maven 3.8+)
- [ ] Clone repository
- [ ] Follow [DEVELOPMENT.md](DEVELOPMENT.md) IDE setup
- [ ] Run `mvn clean javafx:run` to start
- [ ] Run `mvn test` to verify setup
- [ ] Read [ARCHITECTURE.md](ARCHITECTURE.md) for code understanding

##  How to Use This Documentation

### Online Reading

1. Browse files on GitHub (recommended for new users)
2. All files are formatted with Markdown
3. Use GitHub's table of contents feature (right side)
4. Search with Ctrl+F or Command+F

### Offline Reading

1. Clone repository: `git clone https://github.com/KumarSrinidhi/exifcleaner.git`
2. Open files in text editor or IDE
3. Search with your editor's find feature
4. Reference as you code

### Adding to Your IDE

Most IDEs can:
- Create local documentation folders
- Integrate with Java help system
- Generate Javadoc from code + documentation
